## Core Algorithms

### LightGBM (Leaf-wise Asymmetric Trees)
- **GBDT / DART / GOSS / RF** — 4 boosting types fully implemented
- Histogram-based split finding (configurable max_bin)
- L1/L2 regularization (lambda_l1, lambda_l2)
- 8 objectives: Regression, Binary Classification, Huber, Poisson, MAE, Quantile, Tweedie, Gamma
- 11 metrics: RMSE, MAE, MSE, MAPE, R2, LogLoss, AUC, Accuracy, F1, Precision, Recall
- Feature Fraction, Bagging Fraction, Path Smoothing, Monotone Constraints
- GOSS: Gradient-based One-Side Sampling (top_rate + other_rate)
- DART: Dropout regularization for trees
- RF: Bootstrap sampling mode

### CatBoost (Symmetric/Oblivious Trees)
- **Ordered Boosting**: Permutation-based target leakage prevention (all nodes at same depth share split condition)
- **Ordered Target Encoding**: Leak-free categorical encoding using only preceding samples
- Bayesian Bootstrap (bagging_temperature)
- Langevin Boosting (stochastic gradient perturbation)
- Model Shrink Rate (per-iteration learning rate decay)
- Random Strength (split score perturbation)
- RSM (Random Subspace Method = Feature Fraction)

### Ensemble Methods
- **Weighted Blending**: `y = w * CatBoost(x) + (1-w) * LightGBM(x)`
  - Automatic optimal weight search via 101-step grid search
- **OOF Stacking (Out-Of-Fold)**: K-Fold cross-validation + Ridge Regression meta-learner
  - Leak-free OOF prediction generation
  - `y = w1*LGB + w2*Cat + bias` (L2-regularized linear model)

## Why Heterogeneous Ensemble?

| Property | LightGBM | CatBoost |
|----------|----------|----------|
| Tree Structure | Leaf-wise asymmetric (deep, complex) | Symmetric oblivious (simple, regularized) |
| Split Strategy | Best-first leaf with max loss reduction | All nodes at same depth share same split |
| Overfitting | Captures complex patterns, overfitting risk | Structural regularization suppresses overfitting |
| Categorical | Requires integer encoding | Ordered Target Encoding built-in |
| Target Leakage | Full-data residual computation | Ordered Boosting prevents leakage |

**Different decision boundaries cancel individual prediction errors — the ideal heterogeneous combination.**
