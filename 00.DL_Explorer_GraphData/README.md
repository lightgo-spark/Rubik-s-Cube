## Overview

Deep Learning Explorer Pro is a professional-grade, standalone desktop application for exploring and visualizing deep learning concepts interactively. No internet connection required. No installation needed — just run the `.exe`.

---

## Features

| Tab | Description |
|-----|-------------|
| Dashboard | Overview of all 25 interactive modules |
| Neuron Sim | Real-time perceptron weight/bias visualization |
| Activations | Compare Sigmoid, ReLU, Tanh, GELU and more |
| Loss Functions | Explore MSE, MAE, Huber, Cross-Entropy |
| Optimization | Visualize SGD, Adam, RMSprop gradient descent |
| NN Lab | Configure layers, train in real-time, view decision boundary |
| CNN | Convolution, feature maps, max pooling visualization |
| Regression | Linear, Polynomial, Ridge, Lasso with residual plot |
| Classification | Logistic, KNN, Naive Bayes with K-Fold CV |
| AutoML | Automated model selection and hyperparameter search |
| XAI | Feature importance, SHAP, PDP explainability |
| NLP | Tokenization, TF-IDF, sentiment analysis |
| Time Series | Moving average, seasonal decomposition, forecasting |
| Anomaly | Z-Score, IQR, Isolation Forest detection |
| Model Compare | Accuracy, F1, AUC, ROC curve, Parallel Coordinates |
| Hyperparameter | Grid search, learning curves |
| Data Analysis | CSV load, correlation matrix, PCA, t-SNE |
| Transformer | Self-Attention mechanism visualization |
| RL Agent | Q-Learning GridWorld simulation |
| LR Finder | Automatic optimal learning rate search |
| GAN Studio | Generative Adversarial Network simulation |
| Word Embed | Word vector 2D visualization and analogy |
| Batch Norm | Distribution normalization visualization |
| Pipeline | Data→Training→Evaluation workflow builder |
| Settings | Theme, font size, data export |

---

## Getting Started

1. Double-click `DL_Explorer_Pro.exe` to launch
2. Use the left sidebar to navigate between modules
3. Adjust parameters using sliders and buttons
4. Press **F1–F5** to quickly switch between key tabs
5. Press **Space** to start/stop training in NN Lab

---

## Data Export

- Experiment logs can be exported from the **Settings** tab
- Exported files use `.dlexp` format (encrypted)
- An integrity log `dl_integrity.log` is created on first launch

- No additional runtime or installation required
